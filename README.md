## ⚙️ Run Locally

Clone the project

```bash
git clone https://github.com/Neel092/Portfolio.git
```

Go to the project directory

```bash
cd Portfolio
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```
